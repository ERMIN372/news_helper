# Python News Aggregator

Масштабируемый скрипт для сбора новейших и самых интересных новостей по вашим интересам.
Интегрирует NewsAPI для сбора статей и Telethon для чтения Telegram-каналов напрямую (без бота).

## Требования

- Python 3.8+
- Установленные библиотеки из `requirements.txt`

## Установка

1. Создайте виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # Для Linux/Mac
venv\\Scripts\\activate     # Для Windows
```

2. Установите зависимости:
```bash
pip install requests python-dotenv telethon
```

3. Настройте файл переменных окружения:
- Скопируйте `.env.example` в `.env`: `cp .env.example .env`
- Получите API-ключ для [NewsAPI](https://newsapi.org/)
- Получите `api_id` и `api_hash` для Telegram на [my.telegram.org](https://my.telegram.org/apps)
- Впишите ваши данные в `.env`

## Запуск скрипта

Запустите скрипт командой:
```bash
python news_aggregator.py
```

При первом запуске Telethon может запросить код подтверждения, который придёт в ваш аккаунт Telegram (или по СМС), а также пароль от двухфакторной аутентификации (если включен).

## Как это работает?

- Скрипт синхронно собирает новости из **NewsAPI** по ключевым словам для: IT/AI, Политики, Авто, HR, Рок/Метал музыки и Ритейла/АЗС (Нефтьмагистраль).
- Асинхронно подключается к **Telegram** через Telethon, обходит список каналов и забирает последние посты.
- Оценивает и фильтрует уникальные новости, собирая **Топ-10 дайджест**.
- Выводит результат в консоль и сохраняет в формате JSON (например, в `public/news_digest.json`), откуда фронтенд (React сайт) может их прочитать.
